# Aditi Simha Portfolio Website

This is a professional, responsive portfolio website for Aditi Simha.

## About

Aditi Simha is a Software Engineer with experience designing and delivering cloud-native microservices, production REST APIs, real-time data pipelines, backend modernization projects, and enterprise-grade distributed systems.

## Included Files

- `index.html` - single-file responsive portfolio website
- `Aditi_Simha_Resume.pdf` - resume file
- `README.md` - project notes

## How to Run Locally

1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

You can also open `index.html` directly in a browser.

## GitHub Pages Deployment

Upload these files directly to the root of the GitHub Pages repository:

```text
index.html
Aditi_Simha_Resume.pdf
README.md
```

Then go to:

```text
Settings > Pages
```

Make sure the source is:

```text
Deploy from a branch
Branch: main
Folder: /root
```

## Notes

The portfolio is built as a single-page website with inline CSS and JavaScript to avoid broken stylesheet paths on GitHub Pages.
It is responsive for mobile, tablet, and desktop views.
